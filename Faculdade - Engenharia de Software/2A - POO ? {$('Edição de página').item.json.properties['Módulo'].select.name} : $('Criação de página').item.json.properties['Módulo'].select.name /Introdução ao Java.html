Introdução ao Java

☕ Por que estudar Java?

Java é uma linguagem de programação robusta e versátil, sendo uma excelente base para aprender outras linguagens.

✔ Popularidade: Comunidade ativa e vasta gama de recursos online.

✔ Plataforma Independente: Funciona em qualquer sistema operacional.

✔ Mercado de Trabalho: Alta demanda por desenvolvedores Java.

🔄 Evolução da Linguagem Java

Java é atualizado a cada seis meses, garantindo melhorias contínuas.

🔹 Desenvolvido pela Oracle.

🔹 Adaptação às demandas do mercado.

🔹 Compatibilidade com versões anteriores.

🏗 Organização e Estrutura de uma Classe Java

📌 Uma classe Java é definida com a palavra-chave class.

📌 Seu nome deve começar com letra maiúscula.

📌 O método main() é o ponto de entrada do programa.

public class MinhaClasse {

public static void main(String[] args) {

System.out.println("Olá, Mundo!");

}

}

📝 Comentários e Documentação em Java

✏ Tipos de Comentários:

// Comentário de uma linha

/* */ Comentário de múltiplas linhas

/** */ Javadoc (usado para gerar documentação de APIs)

📌 Nomenclaturas e Convenções em Java

✅ Pacotes: Sempre em minúsculas (meu.pacote).

✅ Classes: Começam com maiúscula (MinhaClasse).

✅ Métodos e variáveis: Iniciam com minúscula (meuMetodo).

✅ CamelCase: Cada palavra começa com maiúscula, exceto a primeira (minhaVariavel).

🚀 Criando um Programa Java Simples

Pacotes ajudam a organizar o código.

package meu.pacote;

public class MinhaClasse {

public static void main(String[] args) {

System.out.println("Hello, World!");

}

}

🔹 Palavras-chave importantes:

package: Organização do código.

public class: Declaração de classe.

public static void main(String[] args): Ponto de entrada do programa.

⚙ Entendendo Métodos em Java

📌 O que são métodos?

Representam ações em uma classe.

Possuem nome, parâmetros e tipo de retorno.

🔹 Exemplo de método com parâmetros e retorno:

public int somar(int a, int b) {

return a + b;

}

📊 Tipos de Dados em Java

🔹 Primitivos:

✔ Inteiros: byte, short, int, long

✔ Ponto flutuante: float, double

✔ Caractere: char

✔ Booleano: boolean

🔹 Exemplo:

boolean ativo = true;

char letra = 'A';

🖥 Exibindo Valores com System.out.println()

O comando System.out.println() exibe mensagens no console.

int numero = 10;

System.out.println("O valor é: " + numero);

🔄 Ciclo de Vida de um Programa Java

1️⃣ Criar o arquivo .java

2️⃣ Compilar com javac (gera .class)

3️⃣ Executar com java (JVM roda o código)

javac HelloWorld.java

java HelloWorld

🔑 Principais Termos Java

✅ JDK (Java Development Kit) → Ferramentas para desenvolvimento.

✅ JRE (Java Runtime Environment) → Ambiente de execução.

✅ JVM (Java Virtual Machine) → Executa o código compilado.

🌟 Vantagens da Linguagem Java

✔ Independência de Plataforma – "Escreva uma vez, execute em qualquer lugar".

✔ Gerenciamento Automático de Memória – Garbage Collector embutido.

✔ Multithreading – Suporte nativo para tarefas simultâneas.

✔ Orientação a Objetos – Permite reutilização e manutenção de código.

📂 Estrutura de um Programa Java

package meu.pacote;

public class HelloWorld {

public static void main(String[] args) {

System.out.println("Hello, World!");

}

}

🔹 Componentes:

✔ package → Define um pacote para organização.

✔ public class → Define uma classe principal.

✔ main() → Método inicial do programa.

✔ System.out.println() → Exibe mensagens no console.
