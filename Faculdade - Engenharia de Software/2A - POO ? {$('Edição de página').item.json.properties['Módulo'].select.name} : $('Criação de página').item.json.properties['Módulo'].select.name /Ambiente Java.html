🌟 Introdução ao Ambiente Java e Configuração Inicial

🖥️ Entendendo o Java

Para executar programas Java, é necessário instalar o Java Runtime Environment (JRE), que fornece a Máquina Virtual Java (JVM) e bibliotecas essenciais.

Desenvolvedores utilizam o Java Development Kit (JDK), que inclui o JRE, compilador e depurador.

📥 Download e Instalação do JDK

🔗 Baixe o JDK pelo site da Oracle: Clique Aqui.

✔ Escolha a versão correta para o seu Sistema Operacional (Windows/Linux/Mac).

📌 Após a instalação, configure as variáveis de ambiente para garantir que o sistema encontre os arquivos do Java.

🔧 Configuração de Variáveis de Ambiente

📍 Para garantir que o Java funcione corretamente, é necessário configurar o Path nas variáveis de ambiente.

Como configurar no Windows 🖥

Vá até Painel de Controle > Sistema > Configurações Avançadas.

Clique em Variáveis de Ambiente.

Encontre a variável Path e edite-a, adicionando o diretório de instalação do JDK.

Confirme e reinicie o Prompt de Comando para aplicar as mudanças.

✅ Para verificar a instalação, execute:

java -version

📄 Criando e Executando um Programa Java

✍ Criando o arquivo Java

Abra um editor de texto e escreva o código-fonte Java.

Salve o arquivo com a extensão .java.

O nome do arquivo deve ser exatamente o mesmo da classe pública.

Exemplo:

public class HelloWorld {

public static void main(String[] args) {

System.out.println("Hello, World!");

}

}

⚙ Compilando e Executando o Código

Compilação

No terminal ou prompt de comando, navegue até a pasta onde o arquivo .java foi salvo e execute:

javac HelloWorld.java

Isso criará um arquivo HelloWorld.class contendo o bytecode Java.

Execução

Para rodar o programa, execute:

java HelloWorld

A saída será:

Hello, World!

🎯 Trabalhando com IDEs (Ambientes de Desenvolvimento)

🔍 O que são IDEs?

IDEs são ferramentas que facilitam a escrita, organização e execução de código Java.

Principais IDEs para Java

✅ Eclipse – Grátis, possui diversos plugins e suporte ativo da comunidade.

✅ NetBeans – Simples e leve, ótimo para iniciantes.

✅ IntelliJ IDEA – Tem versão gratuita e uma paga, com funcionalidades avançadas.

🚀 Instalando e Configurando o Eclipse

Passo a passo

🔹 Baixar o Eclipse: Clique Aqui

🔹 Descompactar o arquivo ZIP e executar o eclipse.exe.

🔹 Definir um Workspace, onde os projetos serão salvos.

🔹 Criar um novo projeto Java em File > New > Java Project.

🏗 Criando um Projeto Java no Eclipse

1️⃣ Criar o Projeto

Vá em File > New > Java Project.

Defina um nome para o projeto e selecione a versão do JRE.

Clique em Finish.

2️⃣ Criar a Classe Principal

No diretório src, clique com o botão direito e selecione New > Class.

Defina o nome da classe e marque a opção public static void main(String[] args).

3️⃣ Escrever o Código

Digite ou cole o código Java no editor do Eclipse.

4️⃣ Executar o Programa

Clique com o botão direito no arquivo HelloWorld.java e selecione Run As > Java Application.

A saída aparecerá no console do Eclipse.

✨ Dicas Extras

✔ Atalhos úteis no Eclipse:

Ctrl + S → Salva o código.

Ctrl + Espaço → Sugestões automáticas de código.

Ctrl + Shift + O → Importa pacotes automaticamente.

✔ Depuração no Eclipse:

Use breakpoints para analisar a execução passo a passo.

Acesse o modo Debug para rastrear variáveis e fluxo do programa.

📌 Executando Java no Linux

Passo a passo

1️⃣ Criar um diretório:

mkdir ~/javadevelopment

cd ~/javadevelopment

2️⃣ Criar um arquivo .java:

nano HelloWorld.java

3️⃣ Compilar e executar:

javac HelloWorld.java

java HelloWorld

🎉 Saída esperada: Hello, World!
