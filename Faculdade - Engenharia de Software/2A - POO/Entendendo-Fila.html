🚀 Introdução à Estrutura de Dados: FILA

A FILA é uma estrutura de dados linear que armazena informações de forma homogênea ou heterogênea. Seu acesso ocorre apenas pelas extremidades, seguindo o princípio FIFO (First-In, First-Out), onde o primeiro elemento inserido é o primeiro a ser removido.

🔄 Operações da Fila

Para manipular uma fila, as operações básicas são:

Enfileirar:

Insere um elemento no final da fila.

Desenfileirar:

Remove o elemento do início da fila.

Fila Vazia:

Verifica se a fila está vazia.

Fila Cheia:

Verifica se a fila atingiu sua capacidade máxima.

Elemento do Início:

Exibe o primeiro elemento sem removê-lo.

Mostrar Fila:

Lista todos os elementos presentes na fila.
