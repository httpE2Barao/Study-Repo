Entendendo Árvores

🌳 Estruturas de Dados: Árvores Binárias

🌿 Introdução

Árvores são estruturas de dados:

✅ Não lineares (dados hierárquicos).

✅ Bidimensionais (relações pai-filho).

✅ Eficientes para buscas e inserções (quando balanceadas).

# Exemplo de nó em Python

class No:

def __init__(self, valor):

self.valor = valor

self.esquerda = None

self.direita = None

🌳 Tipos de Árvores

1. Árvore Binária Comum

Grau máximo: 2 filhos por nó.

Uso: Estruturas hierárquicas simples.

3. Árvores Balanceadas

⚠️ Árvore Degenerada:

Forma uma lista encadeada (ineficiente).

Exemplo: Inserir 1 → 2 → 3 → 4 em uma BST sem balanceamento.

🔧 Operações Básicas

🔄 Travessias (Traversals)

// Exemplo de em-ordem em Java
void emOrdem(No no) {
    if (no != null) {
        emOrdem(no.esquerda);
        System.out.print(no.valor + " ");
        emOrdem(no.direita);
    }
}

➕ Inserção

Sempre nas folhas.

Complexidade: O(log n) (balanceada) ou O(n) (degenerada).

➖ Remoção

Caso 1: Nó folha → Exclusão direta.

Caso 2: Nó com um filho → Substitua pelo filho.

Caso 3: Nó com dois filhos → Substitua pelo sucessor em-ordem.

💡 Aplicações Práticas

Bancos de dados: Índices (ex: B-Trees).

Sistemas de arquivos: Diretórios.

Inteligência Artificial: Árvores de decisão.

Compactação: Algoritmo de Huffman.
