Programação Orientada a Objetos em Java

Visão Geral 🌟

Classes: São como plantas de um projeto, definindo os atributos (características) e métodos (ações) dos objetos.

Objetos: São as instâncias criadas a partir das classes, representando elementos reais do programa.
