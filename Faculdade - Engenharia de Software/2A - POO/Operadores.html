Operadores

💻 Introdução aos Operadores e Tipos de Dados em Java 🛠️

Java é uma linguagem de programação fortemente tipada, o que significa que precisamos definir o tipo de dado ao declarar uma variável. Isso garante eficiência e precisão nas operações realizadas. Vamos explorar os principais operadores e tipos de dados em Java de maneira prática e visual! 🌟

🧮 Tipos de Dados Primitivos em Java

Java oferece diversos tipos primitivos para armazenar diferentes tipos de informações:

byte: números inteiros (-128 a 127)

short: números inteiros (-32.768 a 32.767)

int: números inteiros (até ~2 bilhões)

long: números inteiros muito grandes (necessita do sufixo L)

float: números decimais (necessita do sufixo F)

double: números decimais com maior precisão

boolean: verdadeiro (true) ou falso (false)

char: caracteres únicos (ex.: 'A')

🔹 Exemplo de declaração:

int idade = 25;

double altura = 1.75;

boolean ativo = true;

char letra = 'J';

🛠️ Operadores de Atribuição

Os operadores de atribuição permitem atribuir e modificar valores nas variáveis.

🔹 Atribuição Simples

int x = 10;

String nome = "Abel";

🔹 Atribuição com Expressão

int soma = x + 5;

int produto = x * soma;

⚠️ Casting (Conversão de Tipos)

Em algumas situações, precisamos converter explicitamente o tipo de dado:

int a = 130;

byte b = (byte) a; // possível perda de precisão

System.out.println(b); // Resultado: -126

📍 Explicação: O valor 130 excede o limite do byte (127), então o Java aplica o método de Complemento de Dois.

⚡ Operadores de Atribuição Composta

Esses operadores combinam uma operação aritmética com a atribuição:

🔹 Exemplo prático:

int valor = 10;

valor += 5;

System.out.println(valor); // Resultado: 15

➗ Operadores Aritméticos

Java suporta as quatro operações matemáticas tradicionais, além do operador de módulo (%).

🔹 Exemplos no código:

int a = 15;

int b = 2;

System.out.println(a / b); // Resultado: 7 (divisão inteira)

System.out.println(a % b); // Resultado: 1 (resto da divisão)

🔔 Atenção: Dividir um número inteiro por zero lança a exceção ArithmeticException.

🔁 Operadores de Incremento e Decremento

Estes operadores são atalhos para adicionar ou subtrair 1 de uma variável.

🔹 Exemplos:

int x = 5;

System.out.println(x++); // Resultado: 5 (pós-incremento)

System.out.println(++x); // Resultado: 7 (pré-incremento)

🔍 Diferença:

Pós-incremento: usa o valor atual e depois incrementa.

Pré-incremento: incrementa primeiro e depois usa o valor.

🔍 Operadores Relacionais

Esses operadores comparam valores e retornam true ou false.

🔹 Exemplo prático:

int idade = 20;

boolean maiorDeIdade = idade >= 18;

System.out.println(maiorDeIdade); // true
