Intruções Condicionais

💻 Operadores Relacionais e Estruturas Condicionais em Java

⚙️ Operadores Relacionais

Os operadores relacionais constroem expressões lógicas com resultado booleano: true ou false. São frequentemente utilizados em instruções condicionais como if e while.

🛠️ Operadores básicos:

== Igual

!= Diferente

< Menor que

> Maior que

<= Menor ou igual

>= Maior ou igual

🔍 Exemplos práticos:

int nota = 7;

if(nota >= 7) {

System.out.println("Aprovado");

} else {

System.out.println("Reprovado");

}

🧠 Dica: Esses operadores são essenciais para estruturas de decisão e loops.

🛑 Instrução IF

O if permite executar blocos de código condicionalmente.

📍 Sintaxe básica:

if (condição) {

    // bloco de código
} else {
    // bloco alternativo
}

🛠️ Exemplo:

int idade = 18;

if (idade >= 18) {

System.out.println("Maior de idade");

} else {

System.out.println("Menor de idade");

}

✅ Tipos de IF:

Simples

Com ELSE

Aninhado (nested IF)

IF-ELSE-IF (escada)

📍 Exemplo de IF aninhado:

int nota = 7;

if (nota >= 7) {

System.out.println("Aprovado");

} else {

if (nota >= 4) {

System.out.println("Prova final");

} else {

System.out.println("Reprovado");

}

}

🧠 Boas práticas: Mantenha os blocos aninhados organizados para facilitar a leitura.

🔄 Instrução SWITCH

O switch compara um valor com casos específicos, dispensando múltiplos if-else.

📍 Sintaxe:

switch (expressão) {

case valor1:

        // instruções
        break;
    case valor2:
        // instruções
        break;
    default:
        // instruções padrão
}

⚡ Exemplo:

int dia = 3;

switch (dia) {

case 1:

System.out.println("Domingo");

break;

case 2:

System.out.println("Segunda-feira");

break;

case 3:

System.out.println("Terça-feira");

break;

default:

System.out.println("Dia inválido");

}

🔍 Observações:

O break impede a execução de cases subsequentes.

O default é opcional e executado se nenhum case corresponder.

🔢 Operadores Lógicos (AND, OR, NOT)

✅ Curto-circuito: Os operadores && e || avaliam apenas o necessário.

⚙️ Exemplos:

if ((5 > 3) && (2 < 4)) {

System.out.println("Verdadeiro"); // Executado

}

boolean ativo = true;

if (!ativo) {

System.out.println("Inativo"); // Não executado

}

🧠 Operador Ternário

O operador ternário simplifica decisões em uma única linha.

📍 Sintaxe:

variável = (condição) ? valor_se_verdadeiro : valor_se_falso;

⚡ Exemplo:

int idade = 20;

String mensagem = (idade >= 18) ? "Maior de idade" : "Menor de idade";

System.out.println(mensagem);

🎯 Dica: Use o ternário para expressões simples.

🛠️ Operadores Bitwise

Operadores que manipulam bits diretamente.

🔍 Exemplo:

int a = 5; // 0101

int b = 3; // 0011

int and = a & b; // 0001 (1)

int or = a | b; // 0111 (7)

int xor = a ^ b; // 0110 (6)

int notA = ~a; // 1010 (-6)

System.out.println("AND: " + and);

System.out.println("OR: " + or);

System.out.println("XOR: " + xor);

System.out.println("NOT: " + notA);

🎯 Conclusão

Entender os operadores relacionais e as estruturas condicionais permite desenvolver programas dinâmicos e com lógica sólida. Java oferece diversas ferramentas para tomada de decisão eficiente, desde o clássico if até o poderoso switch.
