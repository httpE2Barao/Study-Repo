Árvores Binárias

🌳 Árvores Binárias de Busca (ABB)

Estruturas eficientes para organização e busca de dados

🌿 Introdução

Árvore Binária de Busca (ABB) é uma estrutura hierárquica onde:

👈 Subárvore esquerda: Valores menores que o nó atual.

👉 Subárvore direita: Valores maiores/iguais que o nó atual.

Objetivo: Permitir buscas eficientes (O(log n) em casos balanceados).

Passo a passo:

# Pseudocódigo de inserção recursiva

def inserir(arvore, valor):

if arvore is None:

return NovoNo(valor)

elif valor < arvore.valor:

arvore.esq = inserir(arvore.esq, valor)

else:

arvore.dir = inserir(arvore.dir, valor)

return arvore

☕ Implementação em Java

Estrutura do Nó:

class BIntNo {

int valor;

BIntNo esq;

BIntNo dir;

public BIntNo(int valor) {

this.valor = valor;

esq = dir = null;

}

}

Classe Principal:

public class ArvoreBinaria {

private BIntNo raiz;

public void inserirNo(int valor) {

raiz = inserir(raiz, valor);

}

private BIntNo inserir(BIntNo no, int valor) {

if (no == null) return new BIntNo(valor);

if (valor < no.valor) no.esq = inserir(no.esq, valor);

else no.dir = inserir(no.dir, valor);

return no;

}

}
