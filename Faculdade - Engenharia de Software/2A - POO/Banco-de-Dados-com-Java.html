🖥️ JDBC: Conceitos e Arquitetura

🤔 O que é o JDBC?

API Java padrão que permite a comunicação entre uma aplicação e diversas bases de dados.

Possibilita a escrita de programas independentes do SGBD utilizado.

📦 Principais Funções do JDBC

Conexão: Estabelecer comunicação com o banco.

Execução: Enviar e executar comandos SQL.

Manipulação: Recuperar e manipular os dados retornados.

Portabilidade: Código que funciona com diferentes SGBDs, desde que o driver esteja disponível.

🏗️ Arquitetura Básica do JDBC

API JDBC: Liga a aplicação ao DriverManager.

DriverManager: Gerencia e escolhe o driver adequado para a conexão.

Driver JDBC: Conecta o DriverManager ao banco de dados.
