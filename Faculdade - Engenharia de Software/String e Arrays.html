String e Arrays

📖 Introdução a Strings e Arrays em Java

Strings e Arrays são fundamentais na manipulação e armazenamento de dados em Java. Vamos explorar conceitos essenciais, boas práticas e exemplos práticos.

📝 Introdução a Strings

Strings são cadeias de caracteres representadas entre aspas, utilizadas para armazenar e manipular texto em Java. Embora pareçam tipos primitivos, Strings são objetos da classe String.

🔍 Exemplos de criação de Strings:

String texto = "Olá, Java!";

String mensagem = new String("Bem-vindo ao mundo das Strings!");

💡 Dica: Utilize equals() para comparar o conteúdo de duas strings.

🔍 Lendo Strings com Scanner

Podemos ler Strings utilizando a classe Scanner com os métodos next() e nextLine().

next(): lê até o primeiro espaço.

nextLine(): lê toda a linha, incluindo espaços.

import java.util.Scanner;

public class LeituraString {

public static void main(String[] args) {

Scanner scanner = new Scanner(System.in);

System.out.print("Digite seu nome: ");

String nome = scanner.nextLine();

System.out.println("Olá, " + nome + "!");

scanner.close();

}

}

📍 Boa prática: Sempre feche o scanner após o uso para evitar vazamentos de recursos.

🎓 Manipulando Strings no Java

Operações comuns:

Concatenação: +

Comparação: equals()

Tamanho: length()

String nome = "Java";

System.out.println(nome + " é incrível!");

System.out.println("Tamanho: " + nome.length());

System.out.println("Comparação: " + nome.equals("java"));

🔑 Importante: equalsIgnoreCase() compara desconsiderando maiúsculas/minúsculas.

🧮 Introdução a Arrays (Vetores)

Arrays são estruturas que armazenam múltiplos elementos do mesmo tipo.

Declaração e inicialização:

int[] numeros = new int[5];

int[] valores = {1, 2, 3, 4, 5};

🔹 Indexação inicia em 0 e vai até length - 1.

Acessando e manipulando elementos:

numeros[0] = 10;

System.out.println(numeros[0]);

⚠️ Erro comum: Acessar índices inválidos causa ArrayIndexOutOfBoundsException.

🛠️ Arrays Multidimensionais (Matrizes)

Matrizes armazenam dados em múltiplas dimensões.

int[][] matriz = new int[3][3];

matriz[0][0] = 1;

System.out.println(matriz[0][0]);

📐 Exemplo de matriz 2x3:

int[][] matriz = {

{1, 2, 3},

{4, 5, 6}

};

🖥️ Exibindo Arrays com java.util.Arrays

A classe java.util.Arrays facilita operações com arrays.

import java.util.Arrays;

int[] valores = {5, 2, 4, 3, 1};

Arrays.sort(valores);

System.out.println(Arrays.toString(valores));

🔍 Dica: Arrays.sort() ordena os elementos em ordem crescente.

🚨 Erros Comuns e Boas Práticas

Esquecer de inicializar arrays.

Acessar índices inválidos.

Comparar strings com == ao invés de equals().

🎯 Pratique sempre!
