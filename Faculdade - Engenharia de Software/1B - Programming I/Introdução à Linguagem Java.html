Exemplo Prático

Um programa em Java que demonstra esses conceitos pode receber dois números inteiros do usuário, calcular o quociente e a potência, e exibir os resultados em uma interface gráfica. Utilizando JOptionPane.showInputDialog() para entrada e JOptionPane.showMessageDialog() para saída, o código oferece uma experiência interativa.

Código Exemplo

import javax.swing.JOptionPane;

public class ProgramaFluxoDados {

public static void main(String[] args) {

int numero1, numero2, quociente;

double potencia;

String mensagem = "";

numero1 = Integer.parseInt(JOptionPane.showInputDialog("Digite o primeiro número inteiro:"));

numero2 = Integer.parseInt(JOptionPane.showInputDialog("Digite o segundo número inteiro:"));

quociente = numero1 / numero2;

potencia = Math.pow(numero1, numero2);

mensagem += "Primeiro número: " + numero1 + "\n";

mensagem += "Segundo número: " + numero2 + "\n";

mensagem += "Quociente da divisão: " + quociente + "\n";

mensagem += "Potência (" + numero1 + "^" + numero2 + "): " + potencia + "\n";

JOptionPane.showMessageDialog(null, mensagem);

System.exit(0);

}

}
