O que é Modularização?

A modularização é uma prática fundamental na programação que consiste em dividir um programa grande e complexo em partes menores chamadas módulos ou subprogramas. Essa abordagem traz inúmeras vantagens:

Facilita a escrita e a leitura do código.

Melhora o processo de teste e depuração.

Reduz a repetição de código, promovendo reutilização.

Torna a manutenção mais simples e eficiente.

Os dois principais tipos de subprogramas são procedimentos e funções. A diferença principal entre eles é:

Funções retornam um valor.

Procedimentos não retornam valores e são utilizados para realizar ações específicas.

Nesta página, focaremos nos procedimentos, explicando suas características e como utilizá-los.

O que são Procedimentos sem Parâmetros?

Os procedimentos sem parâmetros são subprogramas que realizam uma tarefa específica sem receber valores externos para sua execução. Eles são úteis quando as informações necessárias para o procedimento já estão disponíveis como variáveis globais no programa.

Estrutura de um Procedimento

Para declarar um procedimento em pseudocódigo, utilizamos a seguinte sintaxe:

procedimento 

  // Declaração de variáveis locais (se necessário)

  // Bloco de comandos do procedimento
fimprocedimento

Exemplo: Olá Mundo

Um exemplo simples de procedimento que imprime "Olá Mundo" na tela:

procedimento olaMundo

inicio

escreva("Olá Mundo!")

fimprocedimento

Como chamar um Procedimento

Dentro do programa principal, o procedimento pode ser chamado pelo nome:

algoritmo "ExemploProcedimento"

inicio

olaMundo() // Chamando o procedimento

fim

Exemplo Prático: Cálculo de Velocidade Média

Vamos criar um procedimento que calcula a velocidade média de um veículo, dado o tempo e a distância percorrida.

Código em Pseudocódigo:

procedimento calcularVelocidade

var
  tempo: real
  velocidade: real
inicio
  tempo <- tempofinal - tempoinicial
  velocidade <- distancia / tempo
  escreva("Velocidade média: ", velocidade, " km/h")
fimprocedimento

algoritmo "ExemploVelocidade"

var
  tempoinicial, tempofinal, distancia: real
inicio
  tempoinicial <- 8
  tempofinal <- 10
  distancia <- 120
  calcularVelocidade()
fim

Explicação:

A variável tempo é calculada dentro do procedimento.

A velocidade média é calculada e exibida no console.
